[使用说明]Notice
=客户端是读取和存入数据的入口；
=Client is the entry of reading and writing data;
=客户端右下角会显示服务器通信状态，如果有故障，请暂停操作，防止数据丢失。
=The right corner will show the server status, if there is a error ,please stop using.
=数据存储在服务器端，保存成功后，数据会被送往服务器；
=The data are stored in the server end, when saved sucessfully, all the data will be sent to server.
=每次点击按钮后，请等待反应之后再进行下一次点击，
=Don't click too much times, wait it for response.
=频繁多次点击按钮发送请求时会出现严重卡顿，因为服务器端处理数据和网络传输数据都需要时间。
=The server end needs time to process data.
=客户端会在最近一段时间不定期更新，如果有新版本号的版本出现，请更新最新的客户端。
=Keep the client version to latest.
=点保存按钮之前，确认一下你是在新建还是在编辑，一般编辑窗口上方会显示出此时的状态，防止误操作；
=Often check the status wheather you are adding a new record or updating a current one.
=万一误操作，可以重新编辑来修改或者直接将该条记录作废掉（确认确实用不到了）；
=Set the state of the record to be Deleted if you do not want to use it.
=作废的记录并非真正删除，相当于放进了回收站中，管理员会定期清除一次来节省数据库容量，这时候才会真正删除掉；
=No data will be deleted actuallly.
=尽量不要使用特殊符号进行输入，以免处罚隐藏的bug；
=Stay away from the special symbols to avoid unexpected bugs.
